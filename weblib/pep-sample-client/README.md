# WebAssembly library: Sample client web page

This folder contains a sample non-production web page that uses the PEP Weblib.

With the `pepWeblibSampleClient` target, the TypeScript Weblib wrapper plus `.wasm` etc. are bundled by webpack into the `./dist/` folder.

See `./start_dev.sh` for how to run it.

For more info, see the [docs](../../docs/public/development/weblib/developing.md).
